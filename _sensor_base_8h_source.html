59  // This returns the 8-bit code for the current status of the sensor.
60  // Bit 0 - 0=Has NOT been successfully set up, 1=Has been setup
61  // Bit 1 - 0=No attempt made to power sensor, 1=Attempt made to power sensor
62  // Bit 2 - 0=Power up attampt failed, 1=Power up attempt succeeded
63  // - Use the isWarmedUp() function to check if enough time has passed
64  // to be ready for sensor communication.
65  // Bit 3 - 0=Activation/wake attempt made, 1=No activation/wake attempt made
66  // - check _millisSensorActivated or bit 4 to see if wake() attempt
67  // was successful
68  // - a failed activation attempt will give _millisSensorActivated = 0
69  // Bit 4 - 0=Wake/Activate failed, 1=Is awake/actively measuring
70  // - Use the isStable() function to check if enough time has passed
71  // to begin a measurement.
72  // Bit 5 - 0=Start measurement requested attempt made, 1=No measurements
73  // have been requested
74  // - check _millisMeasurementRequested or bit 6 to see if
75  // startSingleMeasurement() attempt was successful
76  // - a failed request attempt will give _millisMeasurementRequested =
77  // 0
78  // Bit 6 - 0=Measurement start failed, 1=Measurement attempt succeeded
79  // - Use the isMeasurementComplete() to check if enough time has
80  // passed
81  // for a measurement to have been completed.
82  // Bit 7 - 0=No known errors, 1=Some sort of error has occurred
83  uint8_t getStatus(void);

87  // Generally, the sensor must be powered on for setup.

103  // This wakes the sensor up, if necessary - that is, does whatever it takes

119  // measurements will be any good. The waits are NOT included in this
120  // function!
121  virtual bool startSingleMeasurement(void);
